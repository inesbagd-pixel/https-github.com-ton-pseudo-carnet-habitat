# Carnet Habitat — Brand System « Le Carnet »

Documentation d'identité visuelle. **Aucun code du site ici** : ce dossier sert uniquement de référence avant l'implémentation.

## Comment le consulter
- **`brandbook.html`** — le mini-brandbook complet (13 pages), à ouvrir dans un navigateur. Défilez : chaque page est une section A4. (Charge Fraunces/Inter via Google Fonts et quelques photos de démo via Unsplash — connexion internet requise.)
- **`pages/page-01.png … page-13.png`** — export statique fidèle, page par page (pour partage / impression hors-ligne).

## Décisions verrouillées
- **Direction** : « Le Carnet ».
- **Sceau** : Cachet-Signet, exécution A (signet plein, pointe ocre).
- **Couleur de marque** : **bleu nuit indigo `#22304C`** + **ocre doré `#C39A3A`**, papier crème. (Le vert sauge initial a été écarté car trop proche d'un autre site / de l'univers déco-lifestyle.)
- **Typographie** : Fraunces (signature) + Inter (lecture) — aucune police ajoutée.
- **Ocre doré** : couleur de signature uniquement (sceau, filets, kickers, folios) ; jamais dominante.

Statut : **identité figée, prête pour l'implémentation par phases** (fondations → marques → icônes → composants → direction artistique).
